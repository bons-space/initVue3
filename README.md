# Vue 3 + Typescript + Vite

This template should help get you started developing with Vue 3 and Typescript in Vite.

## 环境要求

VSCode + Volar

推荐使用setup语法糖

推荐使用pnpm包管理工具

## 命令

全局安装pnpm
sudo npm i -g pnpm

在终端中cd到项目，安装依赖包
pnpm install

本地启动(我只是用pnpm安装依赖包，实际启动其实是vite启动的，包管理用啥都无所谓)
npm run dev

打包
npm run build
